# Compression of boosted decision tree models

This contents of this repository are '.ipynb' notebooks related to exploration and analysis of different approaches/attempts to simplify/optimize and/or compress ML classification models.

## Data

All data used is either synthetically generated for the purpose of this project or publically available in python libraries (such as sklearn.datasets).

## Instructions for editing '.ipynb' files

In order to edit and run '.ipynb' notebooks you need Jupyter Notebook or a similar application/environment (for example Google Colab). At this moment all notebooks are only using standard python libraries.
